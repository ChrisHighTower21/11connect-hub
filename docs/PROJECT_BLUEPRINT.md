# Flutlicht-Helden Hub – Project Blueprint V1.0

## Vision
Statistik- und Managementplattform für EA FC Pro Clubs.

## V1.0 Ziel
Nach einem Spiel können Spielerleistungen erfasst, gespeichert, berechnet und ausgewertet werden.

## Kernstruktur
EA FC Zyklus → Wettbewerb → Saison → Spiel → Spielerleistung

## Pflichtstatistik
V1.0 nutzt den EA-FC-Übersichtsreiter als Pflichtdatenbasis.

## Nicht Teil von V1.0
OCR, Accounts, Discord, erweiterte Reiter, Heatmaps.
